Мультиагентная система генерации и проверки гипотез на основе табличных данных
Описание

Исследовательский проект по разработке прототипа мультиагентной системы для автоматической генерации и проверки статистических гипотез на основе табличных данных.
Архитектура

Система состоит из шести специализированных агентов:

    Data Loader & Preprocessor - загружает и предобрабатывает данные

    Агент генерации гипотез - анализирует данные и формулирует гипотезы

    Агент выбора метода - определяет подходящий статистический метод

    Агент выполнения анализа - проводит статистические тесты

    Агент контроля качества (QA Inspector) - проверяет качество анализа

    Агент интерпретации - формирует выводы на естественном языке
