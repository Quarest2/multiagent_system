# Мультиагентная система генерации и проверки гипотез на основе табличных данных

## Описание
Исследовательский проект по разработке прототипа мультиагентной системы для автоматической генерации и проверки статистических гипотез на основе табличных данных.

## Архитектура
Система состоит из пяти специализированных агентов:
1. **Агент-координатор** - управляет workflow
2. **Агент генерации гипотез** - анализирует данные и формулирует гипотезы
3. **Агент выбора метода** - определяет подходящий статистический метод
4. **Агент выполнения анализа** - проводит статистические тесты
5. **Агент интерпретации** - формирует выводы на естественном языке
